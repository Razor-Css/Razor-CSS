Razor CSS Docs



Why should you choose Razor?

This one is easy for us to answer, we offer quick support and fast servers with a easy to use library that lets you learn more css and html by deepening your understanding of web design and why it is so important. Razor is made for people who need to code large amounts of websites in a short amount of time, therefore we have made razor as easy to use as we can. We are also open to suggestions, see the support and contact section to learn how to contact us regarding suggestions and support.



What you need

Some way to host your website, you can find some really good hosts on our list of ways to host your website here.

Domain or sub domain that points to your website, we also have a list for that here.

Basic front end knowledge, we are going to start making tutorials on front end developement, stay tuned here.



Getting started

First you need to make a HTML file so you can get all of the code somewere.

Then you just need to set up the file, wich, we are taking you know how to do because in the what you need section we said that you need basic front end knowledge.

Now it is time to start coding, you now need to put something from this list of different themes that you can choose from, this site has all of the code that you need to put in the <head> section to link our CSS to your document.

Now you are ready to use the basic elements of Razor!



Basic elements

We have many easy to use elements, you can see here how to make them

Lists

To add a list you just need to get what you want to have in your list and add this code:
<div class="list"> <p>1</p> <p>2</p> ect </div>
