How do i use the icon library?

Add this line to your head code: <link rel="stylesheet" href="https://razorcss.xyz/icons/a6/css/rcss-icons-b6.css">

Go to the demo page and find the icon you want.

Add the icon name in this tag: <i class="INSERT YOUR ICON NAME HERE"></i>
